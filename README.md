# transltr
A Node.js module for transltr widget
## Installation 
```sh
npm install transltr --save
yarn add transltr
bower install transltr --save
```
## Usage
### TypeScript
```typescript
import Tranltr from 'transltr';

function App() {
    return (
        <div>
            <Transltr id='YOUR_CLIENT_ID'>
        </div>
    )
}

```
